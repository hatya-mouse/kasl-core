# 1. Introduction to KASL

Welcome to the documentation for the Knodiq AudioShader Language (KASL).

## What is KASL?

KASL is a domain-specific language (DSL) designed for creating real-time synthesizers and audio effects within the **Knodiq** digital audio workstation (DAW).

Its features are focused on the specific needs of audio processing, providing the necessary tools to build custom instruments and effects.

## Core Features

*   **Readable Syntax**: KASL aims for a clean and readable syntax, allowing you to focus on your audio algorithms rather than complex code structures.

*   **Just-In-Time (JIT) Compilation**: To achieve the performance required for real-time audio, KASL code is compiled to machine code at runtime by a JIT compiler.
